（偏）科研

  出于不能浑浑噩噩度日子的目的，我终于重启了于120天前最后更新的“（偏）科研”这一tag，尽管120天前的那篇post是未完成的（基于懒惰这一因素，就先放在那里😊）。这次计划简单学习一下quantum的相关知识，通过阅读Quantum Computation and Quantum Information这本书（大家都说好😊）（哇，我的倒装句用得很棒）。

  • quantum bit (qubit). qubit has a state，就像bit has states 0 or 1一样。bit只有两种states，但是qubit有很多种states：“The state of a qubit is a vector in a two-dimensional complex vector space.”（我觉得或许应该是The state of a qubit corresponds to a vector in a two-dimensional complex vector space我人傻了对不起。）。 提到vector space那么就会有basis。对于state vector space而言，有两个特殊的states ∣0>|0>∣0>和∣1>|1>∣1>，被称为computational basis states，它们组成了state vector space的一组标准正交基。注意到state vector space是定义在复数域上的，所以state vector space中的vector（也可以成为superposition） 形如(α∣0>,β∣1>)(\alpha|0>,\beta|1>)(α∣0>,β∣1>)（对应的state形如∣ϕ>=α∣0>+β∣1>|\phi>=\alpha|0>+\beta|1>∣ϕ>=α∣0>+β∣1>）形如∣ϕ>=α∣0>+β∣1>|\phi>=\alpha|0>+\beta|1>∣ϕ>=α∣0>+β∣1>，其中α,β\alpha,\betaα,β是复数。对于一个vector space来说，除了标准正交基之外，任意一组线性无关的向量也可以组成一组基，因此，给定任意两个线性无关的state vectors（书中只说了任意两个不同的states：“More generally, given any basis states ∣a>|a>∣a> and ∣b>|b>∣b> for a qubit, it is possible to express an arbitrary state as a linear combination α∣a>+β∣b>\alpha|a>+\beta|b>α∣a>+β∣b> of those states.”，不过我想应该是线性无关的，这一点等明天有空再仔细想，先搁置在这 书中没说“任意两个不同的states”，是我眼瞎了。），也可以组成state vector space的一组基。measure a qubit是指…（好的，这是一个很复杂的东西，等我看完了再写）。虽然quantum measurement是一个很复杂的东西，但是有一句相关的"provided the (basis) states are orthonormal, it is possible to perform a measurement with respect to these two basis states"（(basis)和these two basis states是出于方便起见我改的），所以measurement大致可以理解为从一个superposition到一个basis state。然后，因为measurement涉及到概率，即到一个basis state的概率以及到另一个basis state的概率，所以一般进行measure时，需要对state vector (or, superposition) 的系数做归一化 (normalization)，这样到一个basis state的概率和到另一个basis state的概率之和才是1。所以，为了qubit可以被measured，“Thus, in general a qubit’s state is a unit vector in a two-dimensional complex vector space.”
